# Machine Learning Code Generator

![page](ml-code-logo.png)
Automatic machine learning code generator with lots of options.

While writing this project, I was very inspired by [jrieke's project](https://traingenerator.jrieke.com/) I thought it would be nice to do the same for sklearn.

## [Try Demo!](https://ml-code-generator.herokuapp.com/)

## Usage

```
streamlit run app.py
```

## Requirements

```
streamlit
```
